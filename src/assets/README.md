# 静态资源目录

## 目录结构

```
assets/
├── images/          # 图片资源
│   ├── icons/       # 图标文件
│   ├── logos/       # Logo文件
│   └── backgrounds/ # 背景图片
├── fonts/           # 字体文件
├── data/            # 静态数据文件
│   ├── mock/        # 模拟数据
│   └── config/      # 配置文件
└── docs/            # 文档资源
    ├── images/      # 文档图片
    └── examples/    # 示例文件
```

## 使用规范

### 图片资源
- 使用 WebP 格式优先，PNG/JPG 作为备选
- 图标使用 SVG 格式
- 文件命名使用 kebab-case
- 提供多种尺寸的图片（@1x, @2x, @3x）

### 字体文件
- 使用 WOFF2 格式优先
- 提供完整的字体族（regular, bold, italic等）
- 字体文件应包含中文字符集

### 数据文件
- 使用 JSON 格式
- 遵循统一的数据结构规范
- 包含完整的类型定义

## 优化建议

1. 图片压缩：使用工具压缩图片大小
2. 懒加载：大图片使用懒加载
3. CDN：考虑使用CDN加速静态资源
4. 缓存：设置合适的缓存策略
